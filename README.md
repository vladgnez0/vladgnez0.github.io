# Персональный сайт учителя информатики (teacher-site-v3)

## Как открыть
1) Откройте файл `index.html` в браузере (двойной клик).
2) Для корректной навигации и скачивания материалов лучше запускать через локальный сервер:
   - VS Code → расширение Live Server
   - или `python -m http.server` в папке проекта.

## Где менять данные
Файл: `assets/data.js`
- ФИО, школа, контакты, ссылки
- новости, объявления
- список материалов (PDF) и проекты

## Как заменить фото
1) Поместите фото в `assets/img/` (например, `teacher.jpg`)
2) В `assets/data.js` замените:
   `photo: "assets/img/teacher.svg"` → `photo: "assets/img/teacher.jpg"`

## Материалы
Готовые PDF лежат в `assets/materials/` и уже подключены на страницу «Материалы».
